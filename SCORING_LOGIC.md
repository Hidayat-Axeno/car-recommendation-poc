## Scoring Rubric (1–10 scale)
Every score was derived purely from brochure-stated features. No subjectivity beyond the rubric below.
---
### `score_city`
*How well-suited for urban daily driving*

| Condition                                    | Points  |
| Fuel type is EV                              | +3 base |
| Body length < 4000mm (micro/hatch)           | +2      |
| Turning radius ≤ 4.5m                        | +1      |
| EV or ARAI mileage ≥ 15 kmpl                 | +2      |
| Has connected car (traffic, charging search) | +1      |
| Base (for any car)                           | 3       |

Comet EV scores 9 (EV+compact+tight turning). Gloster scores 3 (large diesel, wide turning).

---

### `score_highway`
*Open road comfort and capability*

| Condition                                 | Points |
| Power > 130PS                             | +2 |
| Torque > 200Nm                            | +1 |
| Has cruise control                        | +2 |
| Has ADAS (adaptive cruise / lane assist)  | +2 |
| EV range > 400km OR fuel tank ≥ 60L       | +1 |
| Base                                      | 4 |

Windsor Essence Pro scores 9 (ADAS+cruise+449km range). Comet EV scores 2 (42PS, no cruise, short range).

---

### `score_family`
*Space, safety, practicality for families*

| Condition                 | Points |
| 7-seater                  | +3 |
| 5-seater                  | +1 |
| Has sunroof               | +1 |
| Boot space > 500L         | +1 |
| 6 airbags                 | +1 |
| Rear AC vents present     | +1 |
| Base                      | 3 |

Hector 7S scores 9. Comet EV scores 3 (4-seater, no boot figure, 2 airbags).

---

### `score_business`
*Premium, executive, chauffeur suitability*

| Condition                             | Points |
| Leather/man-made leather seats        | +2 |
| Screen ≥ 14 inch                      | +2 |
| Has connected car platform            | +1 |
| Has wireless charger                  | +1 |
| Has ventilated seats                  | +1 |
| Automatic transmission                | +1 |
| Base                                  | 3 |

Gloster Savvy 6S scores 10. Comet EV Executive scores 3.

---

### `score_mileage`
*Fuel/energy efficiency*

| Condition | Score |
| EV (any) | 9 base |
| Petrol ARAI ≥ 16 kmpl | 9 |
| Petrol ARAI 14–15.9 kmpl | 7 |
| Petrol ARAI 12–13.9 kmpl | 5 |
| Petrol ARAI < 12 kmpl | 3 |
| Diesel ≥ 18 kmpl | 8 |
| Diesel 14–17.9 kmpl | 7 |
| Diesel < 14 kmpl | 6 |

Comet EV scores 9. Gloster 4x4 (13 kmpl est.) scores 6. Hector CVT (13.5 kmpl) scores 5.

---

### `score_adas`
*ADAS feature richness — this is binary-weighted, not gradual*

| Condition | Score |
|---|---|
| No ADAS at all | 0 |
| Only parking sensors + rear camera | 2 |
| Partial: cruise + some warnings only | 4 |
| Full Level 2 suite (ACC, AEB, LKA, LDW, TJA, FCW all present) | 9 |

Only Hector Savvy Pro, Windsor Essence Pro, ZS EV Essence, Gloster Savvy, Majestor Savvy, and Astor Savvy Pro score 9. Everything else is 0 or 2.

---

### `score_performance`
*Engine/motor output feel*

| Condition | Score |
|---|---|
| Power > 200PS | 10 |
| Power 150–200PS | 9 |
| Power 130–149PS | 7 |
| Power 100–129PS | 6 |

Gloster 4x4 (216PS) scores 9+1=10 cap. Comet EV (42PS) scores 3. Windsor EV (136PS, Automatic) scores 7+1=8.

### `score_tech`

| Condition | Points |
|---|---|

| Voice commands (AI assistant) | +1 |

Windsor Essence Pro (i-SMART + digital key + OTA + voice + wireless carplay + 15.6" + V2L) scores 9. Comet EV Executive (no i-SMART) scores 3.

---

### `score_eco`
*Environmental impact — fixed by fuel type*

| Condition | Score |
|---|---|
| EV (any) | 10 |
| Petrol, displacement < 1.5L | 5 |
| Petrol, displacement ≥ 1.5L turbo | 5 |
| Diesel | 5 |

All EVs get 10. All petrol and diesel get 5. Simple and honest — we don't have CO₂ figures from brochures so we don't pretend to rank within fuel types.

---

### `score_value`
*Features per rupee — most subjective score*

| Condition | Score |
|---|---|
| Entry/base variant of model | 8–9 |
| Mid variant | 6–7 |
| Top variant | 4–5 |
| Adjust +1 if notably over-equipped for price | +1 |
| Adjust −1 if notably sparse for price | −1 |

This is the one score you should revisit most often as prices change. A car that was great value at launch may not be after a price hike.

---
```sql
SELECT
  model_name, variant_name, ex_showroom_price,
  has_adas,    score_adas,
  has_sunroof, score_comfort,
  screen_size_inch, score_large_screen,
  seater,      score_family,
  score_offroad, score_value
FROM mg_variants
ORDER BY model_name, ex_showroom_price;
```
